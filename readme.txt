Common coupon label for all coupon

== Description ==

"WM Coupon Label for WooCommerce" is use for Common coupon code. This is just a title for coupon code. This display after coupon code applied.

If you have an interest in customization this plugin please let me know.

== Installation ==
1. Install the WM Common Coupon Label for WooCommerce
2. Activate the plugin
3. Wordpress Admin > Coupon > Add/Edit
